Length

    Length is used to measure distances.


  • The Base Unit for Volume using the Metric System (SI) is the Meter.

  • Length can describe the dimensions of an object (length, width, height) or can describe the distance between two objects.


Metric System (SI) Units for Length:

1 Kilometer (km) = 1000 Meters (m)
1 Meter (m) = 100 Centimeters (cm)
1 Centimeter (cm) = 10 Millimeters (mm)

Convert U.S. Customary Units to Metric System Units:

1 Inch (in) = 2.54 Centimeters (cm)
1 Foot (ft) = 0.3048 Meters (m)
1 Mile (mi) = 1.60934 Kilometers (km)
