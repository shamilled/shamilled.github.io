Conclusion 1

There are quite a lot of things that we can learn about this simple experiment.

One of the first things we discovered is that having enough cashiers to handle the average customer flow is not enough to always be able to avoid random line backups that lead to customers leaving.

The solution was to hire a few more cashiers than we calculated we would need. This is so we can handle the times that our store is overloaded.

Overloaded means we have more customers than we can handle.
